# Código de conducta

## Nuestro compromiso
Queremos una comunidad abierta y acogedora. Nos comprometemos a que participar en este proyecto sea una experiencia libre de acoso para todas las personas, sin distinción.

## Comportamiento esperado
- Usar lenguaje respetuoso e inclusivo.
- Aceptar críticas constructivas con buena disposición.
- Centrarse en lo que es mejor para las personas afectadas por la emergencia y para la comunidad.
- Mostrar empatía: este proyecto trata sobre personas en situaciones difíciles.

## Comportamiento inaceptable
- Acoso, insultos o ataques personales.
- Publicar información privada de terceros sin permiso.
- Usar los datos del proyecto para fines distintos a la reunificación familiar.

## Aplicación
Los responsables del proyecto pueden eliminar contribuciones o vetar a quienes incumplan este código. Para reportar conductas, contacta a [CANAL DE CONTACTO].
